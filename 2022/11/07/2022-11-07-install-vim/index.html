Vim, 隨時補充

2022-11-07

install vim

用部落格來練練 vim

網路上有許多入門資源，我直接參考 learn-vim

先 install 吧

有兩個資源可以參考：

  1. vim-web-side
  2. vim-github

我覺得 GitHub 比較清楚，依照電腦系統不同，有不一樣的安裝方式，以下是 Mac intel 晶片的安裝方式

  1. 安裝 brew

  2. brew install vim

  3. 安裝結束：安裝的時間滿久的，不確定是我電腦跑得比較慢，還是本身安裝時間就長

簡單指令介紹

  • 開啟檔案 ： vim fileName
  • 退出： :q
  • 存檔： :w

兩個指令可以同時使用，:wq -> 存檔並退出
