HTML 的 head 有些什麼

HTML 的 head 有什麼

HTML 是網頁的架構，由語意標籤組成，這裡不會特別紀錄標籤，會把重點放在 head裡面有些什麼。
在紀錄 head 之前，會先說明 、 、、 是什麼

HTML 有些什麼？


文件的種類，告訴瀏覽器文件的種類是 HTML

為什麼需要告訴瀏覽器文件的種類？ 與瀏覽器渲染畫面的模式有關。

瀏覽器有三種渲染畫面的模式：

  • quirks mode、
  • almost standards mode、
  • standards mode

    確保瀏覽器在渲染畫面的時候是在正確模式下進行

  • 標籤內包含整個頁面的內容
  • 又可以被稱為 root element

標籤內的內容不會顯示給使用者看到，包含 HTML 的文件內容

  • title
  • meta
  • CSS / JavaScript
  • script
